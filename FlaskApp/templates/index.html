Interactive APIs with Flask

Usage Instructions

Route: /convert

This route uses the Pillow package to convert an image from one format to another. To use this route, make a POST request to /convert with the following parameters:

  • image: The image file to be converted.
  • output_format: The desired output format for the image.

The available output formats for images are: JPEG, PNG, and GIF.

Code Example:

curl -i -X POST -F "image=@/Users/liyifan/Desktop/image.png"
                    -F "output_format=desired_format" "http://127.0.0.1:5000/convert"

Route: /download

This route allows you to download a YouTube video to the server using the pytube package. To use this route, make a POST request to /download with the following parameter:

  • video_url: The URL of the YouTube video to be downloaded.

Code Example:

curl -i -X POST -F "video_url=Youtube video URL" "http://127.0.0.1:5000/download"
